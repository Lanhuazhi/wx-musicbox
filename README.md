# 基于Django 搭建的微信小程序

该项目主要是通过 爬虫 + Django +微信小程序 完成的，将爬取的歌曲保存为 json 格式的文件，再读取该文件展示到微信小程序

### 功能

  1、歌曲的获取
  
  2、歌曲的分类
  
  3、播放
  
  4、音乐进度条
  
  5、歌词的同步
